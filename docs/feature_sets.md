# Feature Sets

## S2 only

Sentinel-2 tabular spectral features such as band statistics and spectral indices extracted around monitoring stations.

## Context only

Non-image context variables, including rainfall, ERA5-Land hourly variables, temporal features, and seasonal indicators.

## S2 + Context

Combined Sentinel-2 and context features. This is the main multimodal tabular feature set used in the current model comparison.

## Interpretation

Turbidity and SS are optical-related targets and are the most suitable targets for Sentinel-2-based modeling. DO, BOD, TCB, FCB, and NH3-N are indirect proxy/risk-screening targets.
